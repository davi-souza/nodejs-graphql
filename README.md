# NODEJS with GRAPHQL
## Init
__Build images:__
```
docker-compose build --no-cache
```
__Install node packages:__
```
docker-compose run node npm i
```
__Run containers:__
```
docker-compose up
```
__Create ElasticSearch Data:__
[Create ElasticSearch data](http://localhost:3000/es/create/data)
